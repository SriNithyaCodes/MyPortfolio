Welcome to my personal portfolio repository! This website showcases my journey as a 2nd-year Computer Science Engineering student, my core technical skills, and the web development projects I've built. 

## 🌟 Overview

This portfolio is designed with a modern, glassmorphism aesthetic, featuring dynamic animations, interactive elements, and a clean, user-friendly interface. It serves as my digital resume and project showcase, reflecting my passion for software engineering and web development.

## ✨ Key Features

- **Modern UI/UX**: Built with a sleek design, utilizing glassmorphism cards, glowing backgrounds, and smooth hover effects.
- **Dynamic Content**: Features a typing animation effect in the hero section for a more engaging introduction.
- **Responsive Design**: fully responsive layout ensuring a seamless experience across desktops, tablets, and mobile devices.
- **Organized Sections**:
  - **About Me**: A brief introduction to my academic background and passions.
  - **My Journey**: A timeline view of my educational progress and self-taught experiences.
  - **Core Skills**: Visual representation of my technical toolkit (HTML, CSS, Python, Git, GitHub).
  - **Featured Projects**: Showcase of my recent work, complete with external links to live demos or repositories.
  - **Contact**: Easy-to-use contact information and social links for networking.

## 🛠️ Technology Stack

- **HTML5**: For semantic structure and content organization.
- **CSS3 (Vanilla)**: For styling, custom animations, glassmorphism effects, flexbox/grid layouts, and responsive design.
- **JavaScript**: For DOM manipulation, scroll reveal animations (`hidden` -> visible classes), and the typing effect.
- **Google Fonts**: Utilizing 'Inter' and 'Outfit' for modern typography.
- **Font Awesome**: For clean, scalable iconography.

## 💻 Local Development

To run this project locally on your machine, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SriNithyaCodes/MyPortfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd MyPortfolio
   ```

3. **Open the project:**
   Simply double-click the `index.html` file to open it in your default web browser, or use a local server like VS Code's "Live Server" extension for a better development experience.
